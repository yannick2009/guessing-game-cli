## Installation & Usage

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/guessing-game-cli.git
    ```
2. Change the working directory:
   ```bash
   cd guessing-game-cli
   ```
3. Build the project:
   ```bash
    cargo build --release
    ```
4. Run the executable:
    ```bash
    ./target/release/guessing-game-cli
    ```

Enjoy the game!
